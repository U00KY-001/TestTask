# TestTask
В данном коде реализован методы парсинга, считывания информации с файлов, конвертации текстовых цифр в числа, а также создания отчетов
Большинство методов обернуты в тесты
Исходя из условий задачи, было принято два допущения 1) для файлов набор колонок всегда одинаков
                                                     2) файлы должны быть помешены в каталог "Files" (для упрощения)
Реализована возможность добавления "отчетов" в код (была задумка сделать фичу с выводом нескольких отчетов подряд, но в задании явного требовании не было)
Это одна из возможных реализаций и в ней есть что дорабатывать, но она достаточно точно следует заданию
